Variáveis e Tipos de Dados:
Em JavaScript, as variáveis podem armazenar diferentes tipos de dados, incluindo números, strings, booleanos, arrays, objetos e valores especiais como null e undefined.
Exemplo:
let numero = 10;
let texto = "Olá, mundo!";
let booleano = true;
let array = [1, 2, 3];
let objeto = { nome: "João", idade: 30 };
let nulo = null;
