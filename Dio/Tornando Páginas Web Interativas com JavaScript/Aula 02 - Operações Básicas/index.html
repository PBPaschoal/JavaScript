Operações Básicas

Os operadores aritméticos são utilizados para a realização de cálculos simples em JavaScript. Representam as operações matemáticas básicas de soma, subração, divisão e multiplicação, além de algumas operações especiais, como o módulo e incremento / decremento.

  • Operador | Significado
  • + Soma
  • - Subtração
  • * Multiplicação
  • / Divisão
  • % Módulo
  • ++ Incremento
  • -- Decremento
