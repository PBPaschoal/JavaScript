Condicionais

As estruturas condicionais estão ligadas à tomada de decisão de um algoritmo. Ao utilizar expressões que retornam verdadeiro ou falso, o algoritmo executa o bloco de comandos relativos a este resultado.

Básicas

  • if
  • else
  • switch
  • ternario
    •

    Operadores lógicos

    Operador | Significado

    • && - AND (E)
    • || - OR (OU)
    • == === - IGUAL
    • != ou !=== - DIFERENTE
    • > - Maior que
    • < - Menor que

    Testando no terminal ou navegador

    Abra o Debug console ou o console do Chrome para digitar os testes abaixo:
    1 == 1
    true
    1 == '1'
    true
    1 === '1'
    false
    1 != '2'
    true
    1 != '1'
    false
    1 !== '1'
    true
    1 == 1 && 2 == 2
    true
